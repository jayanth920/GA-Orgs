## Making a templates

We have an `input` element...

```html
<input type="text" placeholder="Your name?">
```

So, let's create a template!

```html
<input type="{{type}}" placeholder="{{placeholder}}" value="{{value}}">
```

We'll save this as a template named `input.hbs`.
