D3 in the console

Open up the JavaScript console and type in the following commands. What do the following methods do? How do they relate to jQuery and/or vanilla JavaScript?

Selecting Elements

  1. d3
  2. body = d3.select("body")
  3. body.append('p').text('Hello d3!')
  4. d3.select("body p").text("Welcome to d3")
  5. body.select("p").text("d3 is cool")

Incorporating Data

  1. body.append('div').attr('class','box')
  2. body.append('div').attr('class','box').text('hi')
  3. body.append('div').attr('class','box').text('hello')
  4. d3.select('.box')
  5. d3.selectAll('.box')
  6. my_data = [20, 7, 32]
  7. d3.selectAll('.box').data(my_data).text( function(d) { return d } )
  8. d3.select('.box')[0][0].__data__
  9. new_data = [10, 50, 25]
  10. d3.selectAll('.box').data(new_data)
  11. d3.selectAll('.box').text( function(d) { return d } )
  12. d3.selectAll('.box').text( String )
