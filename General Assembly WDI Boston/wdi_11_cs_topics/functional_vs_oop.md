# Object Oriented Programming

## Strengths

- Popular
- Conceptually easy at first to learn "A Person can Say a message"
- Provides structure
- Code Reuse and Recycling
- Encapulation

## Weaknesses

- Hard to debug instance variables
- Hard to parallelize
