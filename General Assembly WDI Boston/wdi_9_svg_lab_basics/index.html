Intro to SVG

Basic example

SVG images need to be inside of an svg tag. Additional information such as the width, height, xmlns, version and id are optional. All elements must be properly closed. The upper left corner is 0, 0.

Note, we don't need any additional files or libraries to make this work

Unsure where else to put this, but the 'g' element is used to group things, like a div.
